# Simple Notes (React)

A lightweight retro-themed notes app built with React. Notes are stored **only in your browser** using `localStorage`.

## Features

- Add, edit, delete notes
- Title required; body optional (inline validation)
- Created/updated timestamps
- Sorts by most recently updated
- Persists automatically to `localStorage` (`notes_app.items`)

## Run locally

```bash
npm install
npm start
```

Open http://localhost:3000
